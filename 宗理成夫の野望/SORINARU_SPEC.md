## 主人公

**宗理 成夫（そうり なるお）** 38歳
中小企業の万年平社員。

### 決意までの流れ（オープニングの動機）

```
居酒屋で国会中継を見てモヤモヤ
　↓
スマホで地元ニュースをたまたま見る
「あさひ市議会・汚職事件　世渡巧三議員に談合疑惑」
「市の公共工事入札で談合。浮いた予算の行方は不明」
　↓
「こいつか……」「この町がこうなったのは……」
　↓
幼なじみの商店街が廃業したのも、この汚職のせいだったと気づく
　↓
「俺がこの町を変える」→翌朝退職届
```

**動機の三層構造：**
- 国会への漠然とした怒り（トリガー）
- 地元・世渡への具体的な怒り（燃料）
- 幼なじみ・商店街への個人的な思い（核心）

**伏線：**
- 世渡の汚職の相手が第3章ボス「汚職 満」につながる
- 商店街の廃業は票田幸枝の惣菜屋にも影響していた

## ゲームシステム

### 基本サイクル

```
月曜〜土曜：行動フェーズ
　└ 1日 = 行動ポイント3つを自由に配分
日曜：イベントフェーズ
　└ 妨害 or チャンスイベントが発生
最終日：選挙・勝敗判定
```

### パラメータ一覧

| パラメータ | 初期値 | 上限 | 説明 |
|---|---|---|---|
| 知名度 | 5 | 100 | どれだけ名前が知られているか |
| 支持率 | 10 | 100 | 有権者からの支持（最重要） |
| 政策力 | 3 | 100 | 政策の質・説得力 |
| 体力 | 10 | 10 | 0で強制入院 |
| 資金 | 50 | 999 | 選挙活動に必要な軍資金 |
| SNS影響力 | 0 | 100 | 高いほど拡散力↑・炎上リスク↑ |

### 章追加パラメータ

| 追加タイミング | パラメータ | 説明 |
|---|---|---|
| 2章〜 | 実績 | 次章の初期ボーナスに変換 |
| 4章〜 | 信頼度 | 有力者ごとの関係値 |
| 5章〜 | メディア好感度 | TV・新聞・ネット個別管理 |
| 6章〜 | 派閥ポイント | 各派閥との関係値 |

### 勝敗判定

```
成夫スコア = 知名度×0.3 + 支持率×0.5 + 政策力×0.2
成夫スコア > 相手スコア → 当選・次章へ
成夫スコア ≤ 相手スコア → 落選
```

### 落選時：3アウト制

```
再挑戦可能（同じ章をやり直し）
再挑戦のたびに相手の初期スコア+5（だんだん難しくなる）
3回落選 → ゲームオーバー
```

## 特殊システム

### 体力0：強制入院

```
「過労で倒れた！緊急入院！」
→ 3日間行動不能
→ その間に敵の妨害が1回無条件発動
→ 体力全回復して退院
→ SNSのみ入院中も使用可
```

### 妨害対応フロー（日曜イベント）

```
妨害発生（行動ポイント1消費して対応を選ぶ）
├ その場で会見    → 支持率+10  リスクあり
├ SNSで反撃      → 支持率+8   知名度↑↑  炎上リスク
├ 根回し周平に任せ → 支持率+12  確実  資金-5
├ 演田に任せる   → 支持率+15か大失敗（五分五分）
└ 無視する       → ダメージ継続（翌週も-5）
```

### 炎上システム

```
SNS使用時・メディア対応時に一定確率で発生
→ 支持率-20 ＋ 翌日メディアが叩く

対応選択：
├ 即謝罪  → 支持率-5で収束（早期対応）
├ 釈明投稿 → 支持率±0  長引くリスク
└ 無視    → 炎上継続  毎日支持率-3
```

### 章をまたぐ引き継ぎ

| 要素 | 引き継ぎ |
|---|---|
| 実績 | 次章の初期ボーナスに変換 |
| 仲間の信頼度 | 完全引き継ぎ |
| 資金 | 一部引き継ぎ（上限あり） |
| SNS影響力 | 完全引き継ぎ |
| スキャンダル履歴 | 消えずに残る（重要） |

